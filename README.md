# Keisan Trading

Backend orientado al análisis, almacenamiento y procesamiento de datos de mercado.

## Descripción
Keisan Trading es un proyecto personal enfocado en el diseño de un backend para ingestión, persistencia y consulta de datos financieros.
El objetivo principal es trabajar sobre modelado de datos, manejo de series temporales y arquitectura backend aplicada a análisis de mercado.

El proyecto está pensado como una base sólida y extensible para futuros procesos de análisis, simulación y automatización.

## Tecnologías
- PostgreSQL
- SQL
- Backend orientado a datos

## Funcionalidades principales
- Almacenamiento de datos históricos de mercado
- Modelado relacional de activos y precios
- Consultas para análisis histórico
- Base estructural para análisis y simulaciones posteriores

## Visión y evolución
Keisan Trading está concebido como una plataforma evolutiva.

Como parte de su evolución, se contempla la incorporación de un entorno de experimentación aislado (*sandbox*), separado del núcleo estable del sistema, con el objetivo de explorar de forma controlada:
- Ajustes automáticos de parámetros
- Heurísticas de autorregulación
- Funciones básicas de autoanálisis

Este enfoque permite experimentar y aprender sin comprometer la estabilidad del sistema principal ni afectar datos reales.

## Estado del proyecto
Proyecto personal en evolución (work in progress).

## Objetivo
Proyecto personal orientado a profundizar en bases de datos, análisis de datos financieros y diseño de backends robustos y escalables.
